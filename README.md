# Livesplit.SummaryTracker

A livesplit component to track all summary stats for The Hobbit.
### Please read the installation carefully and follow every step to avoid issues!

# Installation

1. Go to <a  href="https://github.com/Shockster218/LiveSplit.SummaryTracker/releases">releases page</a> and download the latest LiveSplit.SummaryTracker.dll release.</br>

2. Navigate to the folder where you have livesplit installed then find the components folder and drop the downloaded SummaryTracker.dll in there.</br>

3. Open up livesplit. Right click the main window and then click edit layout. On the left, click the + button, find the control option from the list, and then add "The Hobbit - Summary Tracker" component from there.</br>

4. Go into the settings of the component and be sure to set the proper category!</br>

5. Click Ok to save the category, then click Ok again to confirm the layout. It's also recommended to right click livesplit then "Save Layout" so it remembers the category.</br>

6. Next, right click the main window of livesplit and click "Edit Splits". Make sure the Game Name is set to "The Hobbit" and Run Category set to the category set in Step 4.

7. Finally, "activate" the autosplitter (if you haven't already) found just below the information section. Then go into the settings, and check the "Automatically Disable Resets when Game Crashes" so its enabled!
